### HOW THE BOT WORKS
This bot is entirely driven by Twitch chat! Whenever a mod enters a valid `!editcom !teams` command in a CQ streamer's chat the bot will tweet a live game update.

### HOW TO CONTRIBUTE
1. Be a mod for your favourite CQ streamer! If you aren't a mod but still want to contribute, please DM `@ChampQueueWard` with your Twitch username.
2. When a new game starts, use the following link to fill out teams and generate the Twitch command: https://champions-queue-cmd-builder.vercel.app/ (see thread for clickable link)
3. Copy the generated command and paste it into Twitch chat
4. Check `@ChampQueueWard` to see your tweet!

### NOTES
- Please avoid altering the format of generated commands, the bot will likely invalidate it and won't tweet anything
- Only the initial mod command for each game will trigger a tweet; subsequent commands for the same game won't be tweeted

*If you find any bugs or have any feedback please DM `@ChampQueueWard`*
